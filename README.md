# 🌱 EcoTech Login Screen

Uma tela de login moderna com tema **EcoTech**, desenvolvida utilizando **HTML, CSS e JavaScript puro**.
Foco em design clean, experiência do usuário e simplicidade de implementação.

---

## 🚀 Demonstração

Tela de login com:

* Interface moderna (efeito glassmorphism)
* Feedback visual de erro e sucesso
* Simulação de autenticação

---

## 📁 Estrutura do Projeto

```
📦 ecotech-login
 ┣ 📄 index.html
 ┣ 🎨 styles.css
 ┗ ⚙️ script.js
```

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3 (Glassmorphism + Gradients)
* JavaScript (Vanilla)

## ⚙️ Como usar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/ecotech-login.git
```

2. Acesse a pasta:

```bash
cd ecotech-login
```

3. Abra o arquivo:

```
index.html
```
